## 2. Технический стек WordPress

### Обязательные плагины

| Плагин | Назначение |
|---|---|
| WooCommerce (последняя версия) | Основа магазина |
| YooKassa для WooCommerce | Оплата (официальный плагин) |
| WP Mail SMTP | Отправка писем (уведомления о заказе) |
| Yoast SEO | SEO |
| WP Rocket или LiteSpeed Cache | Кеширование |
| Wordfence | Безопасность |

### Тема
Кастомная тема или дочерняя тема на базе **Blocksy** / **Kadence** — с реализацией дизайна по секции 7. Никаких Elementor / WPBakery — только Gutenberg блоки или чистый PHP/CSS.

### Хостинг
- PHP 8.2+, MySQL 8.0+
- SSL обязателен (ЮКасса требует HTTPS)
- Минимум 2 GB RAM

## 4. Каталог

### 4.1 Страница категорий (`/catalog`)

**Структура:**
- Баннер вверху с названием категории + фоновое изображение
- Фильтры (слева или сверху):
  - Категория: 2D / 3D / ГЕККО АРТ
  - Размер: S (А5) / M (А4) / L (А3)
  - Цена: слайдер от/до
  - Сортировка: популярные / новинки / цена ↑↓
- Сетка товаров: 3 колонки (desktop), 2 (tablet), 1 (mobile)

**Карточка товара:**
```
┌─────────────────────┐
│   [Фото товара]     │
│   В наличии ●       │
├─────────────────────┤
│ Название набора     │
│ ◈ 7 700 деталей     │
│ ⏱ 18 часов сборки   │
│ ⟷ 50×70 см          │
├─────────────────────┤
│ 4 990 ₽   [КУПИТЬ] │
└─────────────────────┘
```

### 4.2 Три категории товаров

#### 2D Изображения
Готовые наборы по темам (как на gekkotoys.ru):
- Собаки, Техника, Экзотические животные, Домашние животные
- Дикая природа, Динозавры, Новый год, и др.
- Каждый товар: название, фото, кол-во деталей, размер, цена, кнопка «Купить»

#### 3D Фигуры
- Машины, животные, праздничные наборы
- Аналогичная структура карточки

#### ГЕККО АРТ (кастомный)
- Не стандартный товар WooCommerce
- **Кнопка «Создать свой» → редирект на пикселизатор** `http://91.229.10.93:5000/`
- После пикселизатора пользователь возвращается на страницу оформления заказа (см. секцию 5.3)
- На странице категории — объяснение процесса (3 шага)

## 5. Корзина и оформление заказа

### 5.1 Корзина (`/cart`)

- Список товаров с фото, названием, ценой, количеством
- Кнопки + / − / удалить
- Промокод
- Итого: сумма товаров + доставка (рассчитывается автоматически)
- Кнопка «Оформить заказ»

### 5.2 Оформление заказа (`/checkout`)

**Поля формы:**
```
Контакты:
  • Имя *
  • Email *
  • Телефон *

Доставка:
  • Выбор способа (Курьер / Пункт выдачи / Почта России)
  • Адрес (если курьер)
  • Город *

Оплата:
  • ЮКасса (банковская карта, СБП, ЮMoney, Сбербанк Онлайн)

Комментарий к заказу (необязательно)
```

**После оформления:**
- Email с подтверждением заказа (номер, состав, сумма)
- Страница `«Заказ принят»` с номером заказа и ссылкой на отслеживание

### 5.3 Оформление кастомного ГЕККО АРТ заказа

Отдельный тип заказа — пользователь пришёл с пикселизатора:
- Поле для загрузки схемы/файла из пикселизатора
- Выбор размера: S / M / L
- Цена рассчитывается по размеру
- Далее стандартный чекаут (контакты → доставка → оплата)

## 6. Профиль пользователя (`/account`)

**Минимальный набор (WooCommerce My Account):**

```
Личный кабинет:
  ├── Мои заказы        — список с датой, суммой, статусом, ссылкой на детали
  ├── Адреса            — email (обязательно) + адрес доставки (необязательно)
  └── Выйти
```

**Регистрация:** email + пароль. Автоматически при оформлении заказа (галочка «Сохранить данные»).

**Статусы заказов:**
| Статус | Описание |
|---|---|
| Новый | Заказ принят, ожидает обработки |
| В обработке | Набор собирается |
| Отправлен | Передан в доставку + трек-номер |
| Доставлен | Получен |
| Отменён | Отменён |

## 10. Оплата — ЮКасса

**Плагин:** [YooKassa для WooCommerce](https://wordpress.org/plugins/woo-yookassa/) — официальный.

**Методы оплаты для активации:**
- Банковские карты (Visa, MasterCard, МИР)
- СБП
- ЮMoney (ЯндексДеньги)
- Сбербанк Онлайн
- Оплата при получении (отдельно, без ЮКасса)

**Настройка:**
1. Зарегистрировать юридическое лицо в ЮКасса
2. Получить `shop_id` и `secret_key`
3. Вставить в настройки плагина
4. Настроить webhooks для автообновления статуса заказа
5. Включить онлайн-чеки (54-ФЗ) если требуется

## 11. Дизайн-система (соответствие лендингу)

Магазин должен визуально совпадать с лендингом на React.

### Цвета

| Токен | Hex | Применение |
|---|---|---|
| bg-light | `#faf9f5` | Фон страниц (светлая тема) |
| bg-dark | `#1c1a15` | Фон (тёмная тема) |
| text-primary | `#3d3929` | Основной текст |
| accent | `#c96442` | Кнопки CTA, акценты, значки |
| surface | `#242018` | Карточки в тёмной теме |

### Типографика
- Заголовки: **жирный (900)**, без засечек (Inter / Manrope / system-ui)
- Тело: 400-500, межстрочный 1.6
- Метки разделов: `text-xs font-semibold uppercase tracking-widest color:#c96442`

### Компоненты
- Радиус скругления: `rounded-2xl` (16px)
- Кнопка CTA: `bg-[#c96442] text-white font-bold px-8 py-3.5 rounded-2xl`
- Кнопка вторичная: `border-2 border-[#3d3929]/30 text-[#3d3929]`
- Карточки: `bg-white border border-gray-100 shadow-sm rounded-2xl`

### Тёмная тема
Поддержка тёмной темы через `prefers-color-scheme` или переключатель (как на лендинге).

### Логотип
`https://gekkotoys.ru/img/logoBlack.svg` — без инвертирования.

## 12. SEO и технические требования

- Все страницы: `<title>`, `<meta description>`, Open Graph теги
- ЧПУ (ЧеловекоПонятные URL): `/product/nabor-s-a5`, `/catalog/2d-izobrazheniya`
- Sitemap XML (Yoast SEO генерирует автоматически)
- Robots.txt: закрыть `/wp-admin`, `/wp-login`, корзину и чекаут
- Скорость: изображения через WebP, lazy loading, кеш
- HTTPS обязателен
